# 03 — 示意图：页面 · 会话列表模块

## 功能对齐（需求）

- 新建对话、会话列表（标识 / 预览 / 最近时间）、选中高亮、进入历史继续聊、避免串会话。

---

## ASCII — 侧栏展开

```
+------------------------+
|  [ + 新建对话 ]         |
+------------------------+
|                        |
|   与产品评审纪要        |  <- 标题或首条预览
|   2026-03-29 14:20     |  <- 最近活跃（小字）
|                        |
| * 周报草稿讨论 *       |  <- * = 当前选中（高亮）
|   2026-03-28 09:01     |
|                        |
|   定时任务配置          |
|   2026-03-27 18:44     |
|                        |
+------------------------+
|  (列表可滚动)           |
+------------------------+
```

---

## ASCII — 点击「新建对话」后

```
+------------------------+
|  [ + 新建对话 ]         |
+------------------------+
|   与产品评审纪要        |
|   2026-03-29 14:20     |
|                        |
| * 新会话 (未命名) *    |  <- 新建并选中，主区清空或欢迎态
|   刚刚                 |
+------------------------+
```

---

## ASCII — 与主区关系（数据流概念）

```
  选中 sessionId = A
        |
        v
  主区拉取「会话 A」的消息列表；输入区发送均带 sessionId=A
        |
  切换选中 sessionId = B
        |
        v
  主区切换为「会话 B」消息；输入区 sessionId=B
```
