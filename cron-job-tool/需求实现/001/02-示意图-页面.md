# 02 — 示意图：页面（智能助手主界面）

## 说明

对齐《需求分析》主对话页：会话列表 + 当前会话消息 + 输入区；《技术方案》路由示例 `/chat`、`/chat/:sessionId`。

---

## ASCII — 桌面端整页布局

```
+--------------------------------------------------------------------------------+
|  [Logo]  智能助手                                                   [当前用户]   |
+===================+============================================================+
|                   |  当前会话标题 / 预览                    [可选: 已参考历史]   |
|                   +------------------------------------------------------------+
|    会 话 列 表     |                                                            |
|                   |                    消 息 区 域                               |
|  [ + 新建对话 ]    |                                                            |
|                   |    +----------------+      +---------------------------+    |
|   ..............  |    | 用户              |      | 助手                       |    |
|   : *当前会话* :  |    | ...               |      | ...                        |    |
|   :   会话 B   :  |    +----------------+      +---------------------------+    |
|   :   会话 C   :  |                                                            |
|   ..............  |    (多轮消息自上而下滚动，新消息在下方靠近输入区)              |
|                   |                                                            |
|                   +------------------------------------------------------------+
|                   |  输 入 区 域                                                 |
|                   |  +----------------------------------------------------------+|
|                   |  | 请输入问题...                                       [发送]||
|                   |  +----------------------------------------------------------+|
+===================+============================================================+
```

---

```
| 当前：会话 A                 |
+------------------------------+

|        (消息区域同桌面)       |
|                              |
+------------------------------+
| [........................][发]|
+------------------------------+
```
