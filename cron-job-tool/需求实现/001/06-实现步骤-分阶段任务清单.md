# 06 — 实现步骤：分阶段任务清单

## 按阶段 / 页面 / 模块划分

---

### 阶段 A — 后端（NestJS）


| 序号  | 任务                             | 说明                      |
| --- | ------------------------------ | ----------------------- |
| A1  | 会话：创建、列表、按 id 查询               | 支撑前端列表与路由               |
| A2  | 消息：按会话查询、时间序                   | 支撑消息区拉取                 |
| A3  | 非流式对话：`query` + 可选 `sessionId` | 返回 `answer`、`sessionId` |
| A4  | RAG：轮后写向量、轮前检索 Top-K、默认本会话     | 与《技术方案》一致               |
| A5  | SSE 流式：携带 `sessionId`、新建会话标识   | 响应头或约定首包                |
| A6  | 流结束与落库一致                       | 与《技术方案》§3.5 对齐          |
