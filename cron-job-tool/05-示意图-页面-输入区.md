# 05 — 示意图：页面 · 输入区模块

## 功能对齐（需求）

- 发送问题、加载与失败、防重复提交；流式时可选停止（产品定）。

---

## ASCII — 默认

```
+------------------------------------------------------------------+
| [ 请输入你的问题.............................................. ] |
|                                                        [ 发送 ]  |
+------------------------------------------------------------------+
```

---

## ASCII — 请求中（防双发）

```
+------------------------------------------------------------------+
| [ .......................................................... ]   |
|                                              [ 发送 (禁用) ]     |
+------------------------------------------------------------------+
     或：占位文案「正在处理...」+ 按钮禁用
```

---

## ASCII — 流式生成中（可选「停止」）

```
+------------------------------------------------------------------+
| [ (可选：禁止输入下一问直到本轮结束) ........................ ]   |
|                                    [ 停止 ]  [ 发送 (灰显) ]     |
+------------------------------------------------------------------+
```

---

## ASCII — 发送失败

```
+------------------------------------------------------------------+
| [!] 发送失败，请检查网络后重试              [ 重试 ] [ 关闭提示 ] |
+------------------------------------------------------------------+
| [ 原问题仍保留.............................................. ]   |
|                                                        [ 发送 ]  |
+------------------------------------------------------------------+
```

---

## ASCII — 与 sessionId 绑定（防串会话）

```
  当前选中会话 B
       |
       v
  query + sessionId=B  --->  后端
       |
  切换到会话 A 后，输入区绑定 sessionId=A（草稿是否保留由产品定）
```
